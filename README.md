## Työn aihe

Teen taloyhtiölleni saunavarausjärjestelmän. Tarkoituksena on, että jokainen talon asunto kykenee varaamaan, muokkaamaan ja perumaan varauksiaan, sekä näkemään saunan varaustilanteen tulevaisuudessa.
